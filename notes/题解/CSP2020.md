# CSP-S 2020

## T1 动物园
题目比较简单直接写正解了
饲料种类数和每个二进制位对应的饲料 $p$ 是完全没用的

买饲料相当于取一个二进制位, 考虑怎么样一个二进制位才是可以取的:

1. 这个位在前面的动物中已经被买过了
2. 这个位不属于不对应任何一种饲料

所以只要把所有动物的编号或起来, 就知道那些二进制位已经被买过了, 结果记为 $animal$

然后把所有饲料编号对应的二进制位同样或起来, 可以知道哪些位的对应了饲料, 结果记为 $type$

然后用 $2^k-1$ 异或上 $type$ 就知道哪些二进制位不对应饲料, 结果记为 $not\_take$

把 $not_take$ 和 $animal$ 或起来就知道哪些二进制位是可取的, 记为 $able$, $able$ 二进制中为 $1$ 的个数记为 $p$

总共就有 $2^p$ 个动物是可以选的

所以答案就是 $2^p-n$

最后一个点卡了 unsigned long long, 要开 $\_\_int128$, 卡法是 $n=0, m=0, c=随便一个数, k=64$

```cpp
#include <bits/stdc++.h>

using namespace std;
typedef unsigned long long ull;
__int128 animal;
__int128 type;
int n, m, c, k;

void write(__int128 x) {
  if (x < 0) {
    x = -x;
    putchar('-');
  }
  if (x > 9) write(x / 10);
  putchar(x % 10 + '0');
}

int main(){
    freopen("zoo.in", "r", stdin);
    freopen("zoo.out", "w", stdout);
    scanf("%d%d%d%d", &n, &m, &c, &k);
    for(int i = 1; i <= n; i++){
        ull t;
        scanf("%llu", &t);
        //已经选过的饲料可以选
        animal |= t;
    }
    for(int i = 1; i <= m; i++){
        int p, q;
        scanf("%d%d", &p, &q);
        __int128 t = ((__int128)1 << p);
        type |= t;
    }
    __int128 full = 0;
    for(int i = 0; i < k; i++){
        full |= ((__int128)1 << i);
    }
    //不存在饲料的二进制位可以选
    __int128 not_take = full ^ type;
    __int128 able = not_take | animal;
    //总共有多少个可以选
    __int128 num = 1;
    while(able){
        if(able & 1)
            num <<= 1;
        able >>= 1;
    }
    //减去已经选过的动物
    write(num - n);
    printf("\n");
    return 0;
}
```
